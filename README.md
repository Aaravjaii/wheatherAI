# 🌤️ WeatherAI

This is a simple web-based weather app built with **Python** and **Flask**.  
It shows real-time weather data for any city using the **OpenWeatherMap API**.

---

## 🚀 Features

- Search weather by city name
- Shows:
  - Temperature
  - Humidity
  - Weather condition
  - Sunrise & Sunset (local time)

---

## 🧰 Technologies Used

- Python
- Flask
- geopy
- timezonefinder
- pytz
- OpenWeatherMap API
